## Project Purpose
Export Import Hub is a modern web platform where users can manage exports, browse global products, and import any product into their personal "My Imports" section with one click. It allows users to view, add, update, and delete products while maintaining a real-time inventory.

## Key Features

### User Features
- ✅ User Authentication: Login & Register
- ✅ Google Sign-In Integration
- ✅ Protected Routes for personal data
- ✅ Real-time product import with quantity validation
- ✅ View all products and detailed product information

### Admin / Exporter Features
- ✅ Add, Update, Delete products
- ✅ Manage available product quantities
- ✅ Track imports by users
- ✅ Dark/Light mode support

### UI & UX
- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Smooth animations using AOS
- ✅ Interactive elements (badges, hover effects, cards)
- ✅ Notifications using SweetAlert2

## Features in Detail
1. **Authentication**
   - Register/Login using email and password
   - Google Sign-In
   - Protected routes for user-specific data

2. **Products Management**
   - Add new products with details like name, image, price, origin country, rating, available quantity
   - Update existing products
   - Delete products
   - View all products and latest products

3. **Import System**
   - Users can import products
   - Quantity automatically updates in real-time
   - Form validation for available quantity

4. **UI & UX**
   - Dark/Light mode toggle
   - Animated badges and hover effects
   - Smooth scrolling animations with AOS
